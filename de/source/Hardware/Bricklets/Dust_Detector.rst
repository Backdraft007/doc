.. _dust_detector_bricklet:

Dust Detector Bricklet
======================

.. note::
  Diese Bricklet ist noch in Entwicklung!


Features
--------

* Misst Staubdichte zwischen 0 und 500µg/m³
* Erkennt Partikel ab einer Größe von ~0,5µm
